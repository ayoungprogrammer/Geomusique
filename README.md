# Geomusique

This is my project for the 2014 Music Day Hackathon in San Francisco. Users
can login and save music to locations and play them back anywhere.

http://geomusique.herokuapp.com

## APIs
* Rdio API
* Google Maps API

### Tools
* Node.js
* Mongodb
